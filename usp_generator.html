이미지 일괄 생성기

상품 특징 엑셀을 올리면 특징 개수에 맞춰 USP 이미지를 자동 생성합니다. 가로 1000px 고정, 세로 가변. 다운로드 시 이미지코드별 폴더 안에 usp.jpg로 압축됩니다.

이미지 설정

엑셀 파일을 끌어다 놓거나 클릭해서 선택 .xlsx / .xls / .csv
컬럼 형식: 이미지코드 · 특징1 (특징1 한 셀에 줄바꿈 입력 가능, 빈 칸은 자동 제외)
상품 0건 · 첫 번째 미리보기만 생성
· 각 행 1개 상품 = 이미지 1장. 특징 셀 안에서 줄바꿈한 항목마다 버튼 1개가 생성됩니다.
· 버튼이 좌우 폭을 넘으면 자동 줄바꿈되어 세로 길이가 늘어납니다.
· 설정값은 브라우저에 저장됩니다. 원래 값으로 돌아가려면 "설정 초기화"를 누르세요.
· 본 HTML 도구는 오픈소스로 제공됩니다.
· 사용 라이브러리/폰트: SheetJS(xlsx, Apache-2.0), JSZip(MIT), Pretendard(SIL OFL 1.1).
